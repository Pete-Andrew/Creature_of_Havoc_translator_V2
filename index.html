Creature of Havoc Translator

This is a decoder for the fighting fantasy book Creature of Havoc using the books original rules for decoding. I added an encoder as well.

There are three rules to this code:

1. each vowel has been replaced by the following letter, e.g. b = a, f = e, etc...
2. existing vowels represent spaces
3. actual spaces are random and mean nothing

However, this code replaces ALL consonants that follow vowels with their preceding vowel. This means the decoding isn't perfect but is easier to read then the raw encoded text.

Some example text from the book:

whpojs ijt. Whb tajs i hbp pfnjng. W hbta hb sad jstvr bf devs. Plf bsfe gpdor fl fbsf ethjs iblb cka fyfec vrsf.

The English translation:

Who is it. What is happening. What has disturbed us. Please god release this black eye curse.
